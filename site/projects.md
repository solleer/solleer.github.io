        <h2>C2 Workout PRs</h4>
        <p class="proj-desc">
            As a religious user of the Concept2 Logbook for tracking my rowing progress, I saw a need for a
            system to track progress over time for the same workout. This need lead to the creation of the
            Workout PRs tracker which allows a user to sign into their <a href="https://log.concept2.com">C2 Logbook</a>
            account and add descriptions of workouts to track. Once a user has added a type of workout to track,
            they can see their best score as well as their next five best scores along with a graph of progress over all time.
        </p>
        <a href="https://workoutprs.richardsollee.com">See the site</a>
    </section>
    <section>
        <h2>RP3 Graphing</h4>
        <p class="proj-desc">
            The RP3 dynamic rowing machine has a great interface for viewing data during a workout and even has
            a website that the information can be uploaded to through the app that the machine uses. However, their
            website does not have many options for analysis post-workout beyond the simple average statistics it
            provides. This website allows the user to select a json file that contains the workout information
            (which can be downloaded when signed into the RP3 logbook) and have stroke by stroke analysis on
            all data points. In addition, seeing the force curve of a stroke on the RP3 is a high selling point of
            using the machine and this website allows the user to click on a stroke and see a graph of their
            force curve for that stroke. Another section of the site allows the user to select multiple different
            pieces from the same workout or different workouts and overlay the average force curves to compare
            differences in shape and size.
        </p>
        <a href="https://rp3graph.richardsollee.com">See the site</a>
    </section>
